# 产品版本历史

用户可感知的功能变更。技术细节见 `CHANGELOG.md`。

## [Unreleased]

### Added
- **管理员用户删除**：管理员后台可搜索/查看/删除用户，删除同步清除 Keycloak 记录。
- **用户资料增强**：个人中心展示 `username` + `displayName`（不可编辑），绑定 WoTB 账号可改。
- **QQ 登录优化**：username 使用 QQ 昵称，displayName 通过 JWT 映射。
- **打手关联用户**：打手必须从已有用户注册，创建时自动分配 Keycloak `booster` role。
- **暗色主题**：全站暗色模式生效（[data-theme="dark"]）。
- **打手订单面板**：打手在个人中心可查看自己的活跃分配订单（状态、时间、备注）。

### Changed
- **首页视觉升级**：工具集首页改为游戏工具站风格首屏，强化 WoTBTools 品牌、入口层级和移动端可读性。
- **排行榜视觉升级**：上传区、筛选栏、表格强调色和错误提示统一为新的 Blitz 风格。
- **QQ 登录**：支持 Keycloak 自定义 Identity Provider（聚合登录平台 juhe）。
- **登录按钮**：QQ 登录直接跳转聚合登录平台。

### Fixed
- **前端硬编码文案**：排行榜上传状态/跳过提示/无效文件提示改为三语 i18n。
- **字段选择器遮挡**：回放解析字段选择改为固定浮层，避免覆盖表格字段和工具栏内容。
- **移动端顶栏**：手机宽度隐藏主题分段控件，避免顶栏挤压和右侧裁切。
- **中文乱码**：修复回放/排行榜/陪练/扩展分析页中文乱码。
- **首页国际化**：卡片/标签/页脚三语 i18n，非管理员不显示用户管理入口。
- **上传区 UI**：美化回放解析上传区，修复 SVG 描边样式问题。

## [2.0.0] - 2026-06-29

### Added
- **陪练功能**：玩家提交需求、管理员审核分配、打手管理、需求管理。前端 `/?view=boost`。
- **用户资料系统**：个人中心 `/?view=profile`，展示名可编辑，WoTB 账号绑定（唯一性约束）。
- 首页新增「寻找陪练」卡片。
- 页面加载动画（旋转环 + 品牌名）。

### Fixed
- 已取消/已完成需求不再显示「分配」按钮。
- 管理员设终态时自动清理活跃分配。

## [1.9.0] - 2026-06-28

### Added
- **实时评分扩展页**：独立 `/extended` 入口，不改现有回放解析页。
- **新字段**：`alpha_damage`、`rank`、`potential_damage`/`_supplement`/`_detail`，单场/汇总导出、API、前端三语 label 均已同步。

### Changed
- **评分算法**：按潜在均伤、KAST、全场 impact、AST、多伤率、场均人头加权。

## [1.8.0] - 2026-06-27

### Added
- **Keycloak 认证**：账号登录/登出，check-sso 游客模式。
- **个人中心**：ProfilePage，用户名显示。
- **工具集主页**：卡片式入口、版本历史。

### Changed
- 域名统一为 `wotbtools.com`，移除 `replay.wotbtools.com`。

## [1.7.0] - 2026-06-27

### Added
- **品牌升级**：logo + favicon 全站统一。
- **赞助页面**：暗色卡片式 + 支付宝/微信二维码，三语 i18n。
- **主题按钮 i18n**：auto/light/dark 三档中英俄切换。

### Fixed
- 排行榜卡片点击正确跳转 `?view=leaderboard`。
- 排行榜战斗时间不再显示 1970 年。

## [1.6.0] - 2026-06-26

### Added
- 排行榜新增 `version`（游戏版本号）和 `battle_time` 列。
- 排行榜按车辆筛选：点击车辆名查看该车专属伤害榜。

## [1.5.0] - 2026-06-26

### Added
- **工具集主页** `wotbtools.com`：暗色卡片式入口，未来工具占位。
- **三档主题切换**：auto/light/dark，Cookie 跨子域名同步。
- **伤害排行榜**（在线版）：上传单场回放录入排行，按车辆筛选，仅收录随机战斗。
- **实时 rating**：POST /api/rating 实时计算评分。

## [1.3.0] - 2026-06-24
