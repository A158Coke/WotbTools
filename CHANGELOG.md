# 版本历史

## [Unreleased]

### Added
- 工具集主页：`wotbtools.com` 作为入口，展示回放提取器及未来工具卡片
- 主题切换：亮色/暗色/跟随系统，主页与 replay 通过 `.wotbtools.com` 域名 cookie 全局同步，localStorage 本地回退

### Changed
- 容器拆分：单镜像 → `postgres:18` + `wotb-backend` + `wotb-frontend` 三服务，改前端不触发 Maven 构建

## [1.4.0] - 2026-06-25

### Fixed
- 存活时间推算更准确：解决部分玩家被误判阵亡或观战期间不停止的问题

## [1.3.0] - 2026-06-24

### Added
- 存活时间列：每位玩家的阵亡时刻与存活时长
- i18n 三语支持（简体中文 / English / Русский），页面显示语言可选
- 评分系统：类似 WN8 的表现评分，按车型基准归一化
- 地图名中文化：Excel 导出与前端显示中文地图名
- 前端全新数据网格皮肤：浅色主题、队伍行底色、评分徽章
- 列选择器重做为下拉面板：分组浏览、拖拽排序
- 上传区交互重做：分状态卡片设计，反馈更清晰
- 便携构建：零宿主机依赖，一键打包离线 exe
- Docker 化部署：多阶段构建 + CI/CD 自动发布

### Changed
- API 返回纯英文 key，显示名由前端 / 导出层各自转换

### Removed
- 删除旧版 Python 离线 exe

## [1.0.0] - 2026-06-23

### Added
- 回放解析引擎：解析 .wotbreplay 文件提取战斗数据
- Web API：预览、导出 Excel、列配置查询
- 前端页面：上传回放、预览数据、排序筛选、自定义列、导出下载
- 导出 Excel：单场战斗明细 + 多场聚合汇总
- 离线 exe：无需搭建环境，开箱即用

## [0.1.0] - 2026-06-22

### Added
- 项目初始化
